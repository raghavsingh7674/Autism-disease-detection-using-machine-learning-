 Autism Disease Detection using Machine Learning

This project focuses on detecting Autism Spectrum Disorder (ASD) using machine learning techniques based on screening questionnaire datasets.

  Features
- Data preprocessing and cleaning
- Model training using classification algorithms
- Prediction of autism likelihood
- Dataset visualization and analysis

 Dataset
- Autism Screening Dataset
- Toddler Autism Dataset 

Tech Stack
- Python
- Machine Learning (Scikit-learn)
- Pandas & NumPy
- Data Visualization

  Workflow
1. Data collection
2. Data preprocessing
3. Feature selection
4. Model training
5. Prediction & evaluation

 Future Improvements
- Web interface for prediction
- Deep learning model
- Real-time screening system
